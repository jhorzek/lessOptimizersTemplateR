# lessOptimizersTemplateR

This repository demonstrates the use of lessOptimizers as submodule in R packages. The package may be used as a template to create new R packages. 
A full documentation of the procedure is provided in the [Documentation of lessOptimizers](https://jhorzek.github.io/lessOptimizers/).

When cloning the repository, please make sure to also include the submodule **lessOptimizers** with `git clone --recurse-submodules https://github.com/jhorzek/lessOptimizersTemplateR.git`.
